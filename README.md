# Applied Machine Learning Projects

This repository contains applied Machine Learning and AI projects developed as part of my Master's studies in Business Information Systems.

The projects focus on real-world data modeling, recommendation systems, and business-oriented predictive analytics.

---

## Technologies Used

- Python
- Scikit-learn
- Pandas / NumPy
- Matplotlib / Seaborn
- TensorFlow / Keras (where applicable)
- Jupyter Notebook

---

## Implemented Topics

### Supervised Learning
- Classification (Random Forest, Gradient Boosting)
- Regression Models
- Hyperparameter tuning (GridSearch)
- Cross-validation and model evaluation

### Unsupervised Learning
- k-Means clustering
- PCA dimensionality reduction
- Elbow Method

### Recommender Systems
- Collaborative Filtering
- Matrix Factorization
- Content-based recommendation approaches

- Data preprocessing and feature engineering
- Business dataset modeling
